# Crackers Web (React + Firebase)

Quick starter for a React app (Vite) with Firebase auth.

Getting started:

1. Copy the env example:

```bash
cp .env.example .env.local
# then edit .env.local and paste your Firebase config
```

2. Install dependencies and run dev server:

```bash
cd web
npm install
npm run dev
```

App includes a minimal Google sign-in example in `src/App.jsx`.
